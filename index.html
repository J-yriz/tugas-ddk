Tentang Saya

Selamat datang di halaman utama saya! Nama saya Fajar Aziz Kurniawan, dan saya seorang mahasiswa yang sedang mendalami dunia teknologi dan pengembangan perangkat lunak. Saat ini, saya sedang berkuliah di Universitas Dian Nuswantoro di Semarang, Jawa Tengah, dan saya sangat tertarik untuk mengeksplorasi berbagai bidang dalam teknologi, terutama pemrograman dan pengembangan web.

Seiring berjalannya waktu, saya mulai berfokus pada pengembangan aplikasi berbasis web dan belajar berbagai teknologi modern seperti Next.js, TailwindCSS, Prisma, NestJS, serta berbagai alat dan framework yang membantu dalam pembangunan aplikasi yang efisien dan mudah digunakan.

Saya percaya bahwa dunia teknologi memiliki potensi besar untuk mengubah cara kita bekerja, belajar, dan berinteraksi dengan dunia di sekitar kita. Oleh karena itu, saya terus berupaya untuk meningkatkan keterampilan saya dan berbagi pengetahuan yang saya dapatkan melalui berbagai proyek dan artikel di situs ini.


Proyek Terbaru

Saat ini, saya sedang terlibat dalam beberapa proyek, salah satunya adalah membuat Learning Management System (LMS) seperti Schoology untuk sebuah kompetisi IT di mana saya berperan sebagai Project Manager sekaligus Backend Developer. Proyek ini merupakan kesempatan besar untuk memperdalam keterampilan dalam mengembangkan aplikasi yang bermanfaat di bidang pendidikan.

Selain itu, saya juga terus belajar dan berlatih dalam hal pemrograman web, khususnya dalam menggunakan Next.js, serta memahami dan menerapkan konsep-konsep seperti Object-Oriented Programming dan Functional Programming.


Tujuan Saya

Tujuan saya adalah untuk terus belajar, mengembangkan keterampilan saya, dan berkontribusi pada dunia teknologi dengan menciptakan aplikasi yang dapat memecahkan masalah nyata. Saya ingin membantu mempermudah kehidupan sehari-hari melalui inovasi teknologi, dan suatu hari nanti, saya berharap dapat menciptakan produk atau aplikasi yang bermanfaat untuk banyak orang.


Hubungi Saya

Jika Anda tertarik untuk berkolaborasi, berdiskusi tentang teknologi, atau sekadar ingin berbagi pengetahuan, jangan ragu untuk menghubungi saya. Anda dapat menemukan detail kontak saya di halaman Kontak.
